# Simple Calculator Project with Pytest

هذا المشروع عبارة عن نظام آلة حاسبة برمجية بسيطة مبنية باستخدام لغة **Python**، وتطبيق مبادئ هندسة البرمجيات في الفحص والتوثيق وإدارة النسخ.

## 🛠️ الميزات والعمليات المتاحة
* **عملية الجمع (Addition):** فحص وحساب جمع الأرقام بدقة.
* **عملية الطرح (Subtraction):** فحص وحساب طرح الأرقام بدقة.

## 🧪 بيئة الفحص الآلي (Testing)
تم استخدام إطار العمل **Pytest** لضمان جودة الكود وخلوه من الأخطاء العشوائية (Bug-Free Testing).

### أمر تشغيل الفحص:
```bash
python -m pytest
